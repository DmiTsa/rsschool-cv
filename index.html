Функция преобразует одно из полей объекта, который передается в качестве параметра. Данное поле заполняется массивом рандомных объектов, заданных интерфейсом

                        
    import type { gameSet, level } from "../types"
    
    type fullLevelCreator = (gs: gameSet) => level
    
    const fullLevelCreator = (gs: gameSet) => {
    const { dif, sets } = gs
    const typesTile: number[] = []
    
    for (let i = 0; i < sets; i++) {
    for (let j = 0; j < dif; j++) {
    typesTile.push(i)
    }
    }
    
    for (let i = typesTile.length - 1; i > 0; i--) {
    let j = Math.floor(Math.random() * (i + 1));
    [typesTile[i], typesTile[j]] = [typesTile[j], typesTile[i]];
    }
    
    return {
    ...gs,
    tiles: typesTile.map((tileType, i) => {
    return { id: i, type: tileType, isOpened: false, isEnabled: true }
    })
    }
    }
    
    export default fullLevelCreator

projects

I have two small projects. The first is a guessing game stylized as a super mario game, written in vanilla js. The project is simple - my goal was to learn how to connect js code with DOM elements on the page. Later I complicated the logic a little and added the ability to create “levels” in a separate JS file.

The second project is more complex - match game duo cards. It was supposed to be a matcher with finding pairs of cards, but later I changed the logic and depending on the complexity and level, you can find not only pairs, but also groups of cards. The project is written in React. The goal of the project was to master react in conjunction with redux. Also I rewrote this project using Type Script. At the moment, the project is 80% complete - some design edits and minor bug fixes are still needed.
